全球API适用于每个服务器。它目前包括：[全球链接](/wiki/floodgate/linking/#what-is-global-linking)、[皮肤上传](/wiki/floodgate/features/#what-is-skin-uploading)、通过gamertag获取xuid，以及通过xuid获取gamertag。
除了一些不可识别的指标外，我们不会存储您无法通过全球API访问的任何其他内容。全球API的源代码可在此处获得[这里](https://github.com/GeyserMC/global_api)，全球链接服务器的源代码可在此处获得[这里](https://github.com/GeyserMC/GlobalLinkServer)。

作为API的用户，您可以获取关于之前加入过任何Geyser + Floodgate服务器的Geyser（特别是Floodgate玩家）的各种信息。这包括：
- 从基岩版gamertag到基岩版xuid的转换方法，反之亦然。仅提供Geyser缓存中玩家的信息。<br />
  如果您需要所有玩家的信息，可以使用第三方API，例如https://mcprofile.io/endpoints，它也提供来自Geyser全球API的信息。

- 皮肤信息：每当基岩版玩家通过Floodgate加入时，他们的皮肤会被转换并上传到mineskin。如果您想获取该转换后的皮肤，请在此端点中使用基岩版玩家的xuid来获取纹理ID，或获取base64编码的"value"，您可以直接在例如玩家头部中使用。
