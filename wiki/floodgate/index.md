# 概览
Floodgate 是一个混合模式的服务器插件。它引入了以下新功能：
- 允许客户端使用 Minecraft: Bedrock 账户加入 Minecraft: Java Edition 服务器，无需付费的 Minecraft: Java Edition 账户。
- 能够在 [Java Edition 上查看 Bedrock 玩家的皮肤](/wiki/floodgate/features#what-is-skin-uploading)。
- 允许插件向 [Bedrock 玩家发送表单](/wiki/geyser/forms/)。
- 能够通过 [全局 API](/wiki/api/api.geysermc.org/global-api/) 查询玩家信息。
- 允许 Bedrock 玩家通过 [全局链接](/wiki/floodgate/linking#what-is-global-linking) 或本地链接绑定到 Java 账户。

这是你需要额外安装到 Geyser 之外的组件。与 Geyser 不同的是，Floodgate 只能作为插件安装在 Spigot（包括 Paper 及其分支）、Bungeecord、Velocity 上，或作为 Fabric 服务器的模组安装，并且只能在安装了它的服务器上运行。
