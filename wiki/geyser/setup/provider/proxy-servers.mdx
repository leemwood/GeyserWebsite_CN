import { Versions } from '@site/src/components/Versions'

:::info
- 仅在代理服务器上安装 Geyser！您可以在所有服务器上安装 Floodgate 以获得更好的皮肤支持，
    并为其他插件提供网络范围的 Floodgate API 可用性。
- 网络上的所有服务器都必须接受 <Versions platform="java"/> Java 客户端，因为 Geyser 会模仿一个。  
:::

1. 在上方的下拉菜单中选择您的服务器主机提供商。
2. 从[下载页面](/download)下载适用于您的代理软件的插件。
3. 将 Geyser jar 放入 `plugins` 文件夹，然后重启服务器。
4. 打开您的 Geyser 配置文件，位于 `/plugins/Geyser-xyz/config.yml`，找到以下内容：

    ```yaml
    bedrock: 
        # 将监听连接的 IP 地址。
        # 通常，只有在您想限制哪些 IP 可以连接到您的服务器时才应取消注释并更改此项。
        #address: 0.0.0.0

        # 将监听连接的端口。这是基岩版玩家用来连接到您服务器的端口。
        port: 19132

        # 某些托管服务会在您每次启动服务器时更改 Java 端口，并要求对基岩/UDP 连接使用相同的端口。
        # 此选项使基岩端口在每次启动服务器时都与 Java 端口相同。
        # 此选项仅适用于插件版本。
        clone-remote-port: false
    ``` 
    关键部分是端口。这是基岩版玩家用来连接到您服务器的端口。
    按照提供商选择器中的说明操作 - 更改端口，必要时更改 `clone-remote-port` 和 `address`。
    如果启用了 `clone-remote-port`，基岩端口将被 Java 端口覆盖！   
    **重要**：其他依赖 UDP 端口的服务/插件（如语音聊天或查询）不能与 Geyser 共享端口。

5. 通过在控制台中运行 `geyser connectiontest <ip> <port>` 来验证是否可以进行连接。

:::info
要允许基岩版玩家无需登录付费 Java 版账户即可加入您的服务器，您可以使用 <a href="/wiki/floodgate/setup/">Floodgate</a>。
:::
