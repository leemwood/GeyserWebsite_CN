# 其他项目概览 \{#overview-of-other-projects}

"其他"分类为你提供不属于 Geyser 或 Floodgate 的不同项目的相关信息。
这包括社区项目，也包括 GeyserMC 团队的其他项目，例如 Hurricane 或 Hydraulic。

    const metadata = useDocById("other/" + id)
