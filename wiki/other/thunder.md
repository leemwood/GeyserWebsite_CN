Thunder 是一个 Java 应用程序，用于将简单的 Java Edition 资源包转换为 Bedrock Edition 资源包。

## 什么是 Thunder？ {#what-is-thunder}

Thunder 是一个简单的 Java Edition 资源包转换器，可以生成 Bedrock Edition 资源包，它目前转换的是修改现有原版资源的资源包。

:::caution

不仅这个项目处于早期测试阶段，它**不**转换自定义物品！请查看 [Rainbow](/wiki/other/rainbow) 获取该功能。

:::

## 使用方法 {#usage}

1. 确保您已安装 Java 17 或更高版本，如果您已安装，您应该能够像普通程序一样打开该文件。
2. 使用左侧的按钮选择 Java 版本资源包。成功后，您应该会看到该包的图标。
3. 更改输出包名称。这是显示给 Bedrock 客户端的名称。
4. 点击转换按钮开始转换过程。这一步可能需要一些时间。
5. 完成后，转换后的 Bedrock Edition 资源包将位于您的 Java 包所在的同一位置。
您现在可以在 Bedrock 客户端上使用该包，或将其放入 Geyser 服务器的 `packs` 文件夹中。
